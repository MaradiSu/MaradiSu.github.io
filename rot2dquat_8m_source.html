5 % DQ = ROT2DQUAT(THETA,AXIS) returns the dual quaternion DQ which is the
6 % dual quaternion representation of a rotation of angle THETA and of
7 % axis AXIS.

12 % array (column i represents the rotation axis of the i-th
13 % rotation) where N is the number of rotations.
14 % - DQ is a 8*N array representing the rotation dual quaternion:
15 % Several combinations are possible:
16 % - THETA is a N-vector and AXIS is a 3-vector ==> DQ is a 8*N

21 % angle THETA and of axis AXIS(:,i).
22 % - THETA is a scalar and AXIS is a 3-vector
23 % - THETA is a N-vector and AXIS is a 3*N array ==> DQ is a 8*N
