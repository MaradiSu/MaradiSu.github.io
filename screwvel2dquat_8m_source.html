6 % returns the dual quaternion DQ which is the dual quaternion
7 % representation of a screw motion velocity with an angular speed OMEGA
8 % [deg/s] around an axis OMEGA_AXIS, with a translation speed SPEED
9 % along the line L (which is the screw motion axis line of the screw
10 % motion dual quaternion DQSCREW) orientation, and with an offset
11 % velocity APDOT of the line L. DQVELSCREW is the derivative of
12 % DQSCREW.
13 % - OMEGA represents the angular speed [deg/s]. It is a scalar or a
14 % N-vector (the i-th element represents the angular speed of
15 % the i-th screw motion velocity) where N is the number of
16 % screw motion velocities.
17 % - OMEGA_AXIS represents the instantaneous rotational velocity axis.
18 % It is a a 3-vector or a 3*N array (column i represents the
19 % rotational velocity axis i).
20 % - SPEED represents the translational speed. It is a scalar or a
21 % N-vector (the i-th element represents the translational
22 % speed of the i-th screw motion velocity).
23 % - APDOT represents the offset velocity (orthogonally to the line
24 % L). It is a 3 vector or a 3*N array (column i represents
25 % the offset velocity of the i-th screw motion axis line).
26 % - DQSCREW is a screw motion dual quaternion. It is a 8-vector or
27 % an 8*N array (column i represents screw motion dual
28 % quaternion i).
29 % - DQVELSCREW is a 8*N array representing the screw motion velocity

33 % returns the dual quaternion DQ which is the dual quaternion
34 % representation of a screw motion velocity with an angular speed OMEGA
35 % [deg/s] around an axis OMEGA_AXIS, with a translation speed SPEED
36 % along the line DQLINE (which is the screw motion axis line of the
37 % screw motion dual quaternion generated with THETA, D and DQLINE)
38 % orientation, and with an offset velocity APDOT of the line DQLINE.
39 % DQVELSCREW is the derivative of DQSCREW = SCREW2DQUAT(THETA,D,DQLINE)

47 % DQLINE. It is a scalar or a N-vector (the i-th element represents
48 % the distance D of the i-th screw motion).
