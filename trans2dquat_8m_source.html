5 % DQ = TRANS2DQUAT(VEC) returns the dual quaternion DQ which is the
6 % dual quaternion representation of a translation of vector VEC.
7 % - VEC represents the translation vector. It is a 3-vector or a
8 % 3*N array (column i represents the translation vector of

10 % - DQ is a 8*N array representing the translation dual quaternion.
11 %
12 % DQ = TRANS2DQUAT(AXIS,D) returns the dual quaternion DQ which is the
13 % dual quaternion representation of a translation of distance D and of
14 % axis AXIS.
15 % - D represents the translation distance. It is a scalar or a
16 % N-vector (the i-th element represents the distance of the
17 % i-th translation) where N is the number of translations.
18 % - AXIS represents the translation axis. It is a 3-vector or a 3*N
19 % array (column i represents the translation axis of the i-th
20 % translation) where N is the number of translations.
21 % - DQ is a 8*N array representing the translation dual quaternion.
22 %
