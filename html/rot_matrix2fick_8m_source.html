3 % ROTMATRIX2VFICK transforms a 3*3 rotation matrix into Fick coordinates
4 %
5 % [VFICK,VFICK2] = ROTMATRIX2FICK(ROTMATRIX) returns the Fick
6 % coordinates VFICK and VFICK2 [deg] corresponding to the rotation
7 % matrix ROTMATRIX.
8 % - ROTMATRIX is a 3*3 array or a 3*3*N tensor (in which case,
9 % ROTMATRIX(:,:,ii) is a 3*3 matrix representing the rotation
10 % matrix corresponding to rotation ii) where N is the number of
11 % rotations. ROTMATRIX can also be a 1*N cell and each cell
12 % component ROTMATRIX{1,ii} is the 3*3 rotation matrix of
13 % rotation ii.
14 % The following reference frame must be used for the rotation
15 % matrix! The positive rotation around x corresponds to a
16 % clockwise torsional rotation. The positive rotation around y
17 % corresponds to a downward rotation. The positive rotation
18 % about z corresponds to a leftward rotation.
19 % - VFICK (resp. VFICK2) is a 3*N array (column i represents
20 % rotation i). VFICK(:,i) can be written as VFICK = [H V T] where
21 % (from the subject's perspective):
22 % - H is the horizontal fick angle (positive when to the left)
23 % - V is the vertical fick angle (around the inter-aural axis).
24 % It is positive when down and is comprise between -90 and
25 % 90deg.
26 % - T is the torsional fick angle (around the line of signt). It
27 % is positive when clockwise.
28 % - VFICK2 is the second solution VFICK2 = [H2 V2 T2] and V2 is
29 % comprised between 90 and 270deg.
30 %
31 % Useful reference: T. Haslwanter (1995), Mathematics of three-dimensional
32 % eye rotations, Vision research, 35(12), 1727-39
33 %
34 % See also FICK2ROTMATRIX, DQUAT2ROTMATRIX, ROTMATRIX2DQUAT
