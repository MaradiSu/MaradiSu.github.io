2 
3 % LINEVEL2DQUAT Transforms a line velocity expressed in vector notation
4 % into its dual quaternion representation.
5 %
6 % DQ = LINE2DQUAT(V,R,VP,RP) transforms the line position, specified by:
7 % - the line orientation V
8 % - the position of any point of the line, R.
9 % - the line orientation rate of change, VP
10 % - the velocity component (orthogonal to the line orientation V)
11 % of point P, RP
12 % V does not need to be unitary, but VP must be orthogonal to V. If
13 % RP has a component in the V orientation, it does not matter, since
14 % it does not change the expression of the resulting dual quaternion
15 % DQ.
16 % V,R,VP and RP must have the same size. The inputs (V,R,VP,RP) are
17 % either a vector of size 3 or an array of size 3*N (column i
18 % represents the input component of Line velocity i) where N is the
19 % number of lines. DQ is either a vector of size 8, either an array
20 % of size (8*N) depending on the input format. Each column of DQ
21 % represents the dual quaternion representation of the corresponding
22 % line position velocity.
23 %
24 % See also POS2DQUAT, VEL2DQUAT, LINE2DQUAT
