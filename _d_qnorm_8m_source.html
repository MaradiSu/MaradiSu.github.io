3 % DQNORM dual quaternion norm
4 %
5 % DQN = DQNORM(DQ) returns the norm DQN of a dual quaternion DQ
6 % - DQ is a dual quaternion. It is a 8-vector or a 8*N array (column i
7 % represents dual quaternion i) where N is the number of dual
8 % quaternions.
9 % - DQN is a dual number (DQN = a0 + epsilon*b0 where a and b are real
10 % numbers) representing the dual quaternion norm of DQ. It exists
11 % only if DQ(1:4) is different from 0. It is a 2*N array:
12 % For a column i, the first component is the non-dual part a0 and
13 % the second component is the dual part, b0.
14 %
15 % Useful reference: Kavan L. , Collins S. , Zara J. and O'Sullivan C
16 % (2008), Geometric skinning with approximate dual quaternion blending,
17 % ACM Transactions on Graphics, 27(4), 1-23
18 %
19 % See also QNORM, DQCONJ, DQINV
20 
21 sdq = size(dq);
22 if sdq == [1 8], dq = dq'; sdq = size(dq); end
23 
24 % wrong format

27  '%d rows in the DQ array. It should be 8.',sdq(1));
28 end
29 n = sdq(2);
30 
31 q0 = dq(1:4,:);
32 q1 = dq(5:8,:);
33 normq0 = Qnorm(q0);

37 end
